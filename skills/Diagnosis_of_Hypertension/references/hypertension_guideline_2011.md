# 中国高血压防治指南（2011修订版）核心内容

## 一、高血压的定义与分类

### 1.1 高血压诊断标准

在未使用降压药物的情况下，非同日3次测量血压，收缩压（SBP）≥140 mmHg和/或舒张压（DBP）≥90 mmHg，可诊断为高血压。

**特殊情况：**
- 患者既往有高血压史，目前正在使用降压药物，血压虽然低于140/90 mmHg，也诊断为高血压

### 1.2 血压水平分类

| 分类 | 收缩压（mmHg） | 舒张压（mmHg） |
|------|----------------|----------------|
| 正常血压 | <120 | <80 |
| 正常高值 | 120-139 | 80-89 |
| 高血压 | ≥140 | ≥90 |
| 1级高血压（轻度） | 140-159 | 90-99 |
| 2级高血压（中度） | 160-179 | 100-109 |
| 3级高血压（重度） | ≥180 | ≥110 |
| 单纯收缩期高血压 | ≥140 | <90 |

**注：**
- 当SBP和DBP分属不同级别时，以较高的分级为准
- 单纯收缩期高血压也可按照收缩压水平分为1、2、3级

## 二、高血压的危险分层

### 2.1 影响预后的危险因素

**心血管危险因素：**
1. 年龄：男性≥55岁，女性≥65岁
2. 吸烟
3. 糖耐量受损（餐后2小时血糖7.8-11.0 mmol/L）和/或空腹血糖受损（6.1-6.9 mmol/L）
4. 血脂异常：
   - TC≥5.7 mmol/L（220 mg/dL）
   - LDL-C>3.3 mmol/L（130 mg/dL）
   - HDL-C<1.0 mmol/L（40 mg/dL）
5. 早发心血管病家族史（一级亲属发病年龄男性<55岁，女性<65岁）
6. 肥胖（BMI≥28 kg/m²）
7. 腹型肥胖（男性腰围≥90 cm，女性≥85 cm）

**靶器官损害（TOD）：**
1. 左心室肥厚：
   - 心电图：Sokolow-Lyons>38 mm或Cornell>2440 mm·ms
   - 超声心动图：LVMI男≥125 g/m²，女≥120 g/m²
2. 颈动脉超声：IMT≥0.9 mm或动脉粥样斑块
3. 颈-股动脉PWV≥12 m/s
4. 踝/臂血压指数<0.9
5. eGFR降低（<60 ml/min/1.73m²）或血清肌酐轻度升高：
   - 男性115-133 μmol/L（1.3-1.5 mg/dL）
   - 女性107-124 μmol/L（1.2-1.4 mg/dL）
6. 微量白蛋白尿：30-300 mg/24h或白蛋白/肌酐比≥30 mg/g（3.5 mg/mmol）

**伴随临床疾患：**
1. 脑血管病：脑出血、缺血性脑卒中、短暂性脑缺血发作
2. 心脏疾病：
   - 心肌梗死史
   - 心绞痛
   - 冠状动脉血运重建史
   - 充血性心力衰竭
3. 肾脏疾病：
   - 糖尿病肾病
   - 肾功能受损（血肌酐）：
     - 男性≥133 μmol/L（1.5 mg/dL）
     - 女性≥124 μmol/L（1.4 mg/dL）
   - 蛋白尿（≥300 mg/24h）
4. 外周血管疾病
5. 视网膜病变：
   - 出血或渗出
   - 视乳头水肿
6. 糖尿病：
   - 空腹血糖≥7.0 mmol/L（126 mg/dL）
   - 餐后2小时血糖≥11.1 mmol/L（200 mg/dL）
   - HbA1c≥6.5%

### 2.2 心血管风险分层标准

| 其他危险因素和病史 | 1级高血压 | 2级高血压 | 3级高血压 |
|-------------------|----------|----------|----------|
| 无其他危险因素 | 低危 | 中危 | 高危 |
| 1-2个危险因素 | 中危 | 中危 | 很高危 |
| ≥3个危险因素或靶器官损害 | 高危 | 高危 | 很高危 |
| 临床并发症或合并糖尿病 | 很高危 | 很高危 | 很高危 |

**风险等级说明：**
- **低危**：10年心血管风险<5%
- **中危**：10年心血管风险5%-10%
- **高危**：10年心血管风险10%-15%
- **很高危**：10年心血管风险≥15%

## 三、高血压的评估

### 3.1 全面的病史采集

1. 家族史：高血压、糖尿病、血脂异常、冠心病、脑卒中或肾脏病的家族史
2. 病程：高血压发现时间、血压最高水平、治疗情况、治疗反应
3. 症状及既往史：
   - 继发性高血压的线索（打鼾、肌无力、发作性软瘫、阵发性头痛、心悸、多汗等）
   - 靶器官损害的症状
   - 既往心脑血管疾病、肾脏疾病、糖尿病、血脂异常、支气管哮喘、痛风等病史
4. 生活方式：盐、酒摄入量，吸烟状况，体力活动量，体重变化，睡眠呼吸暂停等
5. 药物应用史：是否服用升高血压的药物（如口服避孕药、甘草、麻黄等）
6. 心理因素：有无精神紧张、焦虑、抑郁等

### 3.2 体格检查

1. 测量血压：双侧上臂血压，必要时测量下肢血压
2. 测量体重指数（BMI）、腰围
3. 检查眼底
4. 检查心脏：心界、心音、心律、杂音
5. 检查血管：颈动脉、肾动脉、腹主动脉、下肢动脉搏动
6. 检查腹部：肾脏是否增大、肿块
7. 检查神经系统

### 3.3 实验室检查

**基本检查项目：**
1. 血生化：
   - 血钾、血钠
   - 空腹血糖
   - 总胆固醇（TC）
   - 低密度脂蛋白胆固醇（LDL-C）
   - 高密度脂蛋白胆固醇（HDL-C）
   - 甘油三酯（TG）
   - 血肌酐、血尿酸
   - 肝功能
2. 尿液分析：尿蛋白、尿糖、尿肌酐
3. 心电图

**推荐检查项目：**
1. 超声心动图
2. 颈动脉超声
3. 餐后2小时血糖
4. 血同型半胱氨酸
5. 尿白蛋白定量（糖尿病患者必查项目）
6. 尿蛋白定量（用于尿常规检查蛋白阳性者）
7. 眼底检查
8. 胸部X线检查
9. 脉搏波传导速度（PWV）
10. 踝/臂血压指数（ABI）

## 四、高血压的治疗

### 4.1 治疗目标

**血压控制目标：**
- 一般高血压患者：<140/90 mmHg
- 老年（≥65岁）收缩期高血压患者：SBP<150 mmHg（如能耐受可降至140 mmHg以下）
- 伴有糖尿病、慢性肾病、冠心病或脑血管病的高血压患者：<130/80 mmHg

### 4.2 非药物治疗（生活方式干预）

1. **减少钠盐摄入**：<6 g/d（普通啤酒瓶盖去掉胶皮垫后水平装满食盐约6 g）
2. **控制体重**：BMI<24 kg/m²；腰围：男性<90 cm，女性<85 cm
3. **不吸烟**：戒烟并避免被动吸烟
4. **限制饮酒**：白酒<50 ml/d、葡萄酒<100 ml/d、啤酒<300 ml/d
5. **体育运动**：中等强度有氧运动，每周4-7次，每次持续30-60分钟
6. **减轻精神压力**：保持心理平衡

### 4.3 降压药物治疗

**降压药物种类：**
1. 钙通道阻滞剂（CCB）
2. 血管紧张素转换酶抑制剂（ACEI）
3. 血管紧张素II受体拮抗剂（ARB）
4. 噻嗪类利尿剂
5. β受体阻滞剂

**降压治疗原则：**
1. 小剂量开始
2. 优先选择长效制剂
3. 联合用药（2级及以上高血压或高危患者起始即可采用联合治疗）
4. 个体化治疗

### 4.4 特殊人群高血压处理

**老年高血压：**
- SBP≥150 mmHg可考虑开始药物治疗
- 目标SBP<150 mmHg，如能耐受可降至140 mmHg以下
- 避免过快、过度降压
- 注意直立性低血压

**妊娠高血压：**
- 血压≥140/90 mmHg需降压治疗
- 目标血压130-155/80-105 mmHg
- 可用药物：甲基多巴、拉贝洛尔、钙通道阻滞剂
- 禁用ACEI、ARB、利尿剂

**儿童青少年高血压：**
- 以百分位法诊断（根据年龄、性别、身高）
- 首选ACEI/ARB或CCB
- 注意继发性高血压筛查

## 五、高血压急症和亚急症

### 5.1 高血压急症

定义：血压严重升高（一般>180/120 mmHg）并伴有进行性靶器官损害。

**常见类型：**
1. 高血压脑病
2. 颅内出血
3. 急性心力衰竭
4. 急性冠脉综合征
5. 急性主动脉夹层
6. 子痫
7. 嗜铬细胞瘤危象

**治疗原则：**
- 静脉降压药物
- 第1小时平均动脉压降幅不超过治疗前水平的25%
- 在2-6小时内将血压降至约160/100 mmHg

### 5.2 高血压亚急症

定义：血压显著升高但不伴有靶器官损害。

**治疗：**
- 口服降压药物
- 24-48小时内逐渐降低血压
- 避免过快降压

## 六、指南要点总结

### 6.1 诊断要点

1. 高血压诊断基于非同日多次血压测量
2. 根据血压水平分为1、2、3级
3. 既往高血压史且正在服药者，即使血压正常也诊断高血压
4. 单纯收缩期高血压按照收缩压水平分级

### 6.2 评估要点

1. 全面评估心血管危险因素
2. 寻找靶器官损害证据
3. 明确有无伴随临床疾患
4. 进行心血管风险分层
5. 筛查继发性高血压

### 6.3 治疗要点

1. 所有患者均应进行生活方式干预
2. 根据心血管风险水平决定启动药物治疗时机
3. 高危和很高危患者应立即开始药物治疗
4. 低危患者可先进行数周生活方式干预
5. 降压目标个体化
6. 优先选择长效制剂，小剂量开始，必要时联合用药
